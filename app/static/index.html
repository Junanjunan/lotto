로또 전략 게임 생성기

저장된 전체 회차를 기준으로 전략별 번호를 생성하고, 게임별 1~5등 분포를 즉시 비교할 수 있는 실험 대시보드입니다.

준비됨

광고

고급 옵션

같은 전략/옵션/게임 수에서 시드를 같게 입력하면 같은 결과를 다시 만들 수 있습니다.

상세 설명

  • 여러 게임을 동시에 만들 때 각 게임 간 번호 겹침을 줄이는 데 집중합니다.
  • 번호 풀이 넓게 퍼져 포트폴리오처럼 분산된 조합을 만들기 좋습니다.
  • 옵션 제약을 많이 켜면 생성 가능한 조합이 줄어들어 실패할 수 있습니다.

상세 설명

  • 각 게임을 독립적으로 무작위 추출하는 기준 전략입니다.
  • 추가 분산 규칙 없이 순수 랜덤 성향을 확인할 때 비교 기준으로 적합합니다.
  • 전략 효과를 비교 실험할 때 가장 중립적인 베이스라인으로 사용하기 좋습니다.

상세 설명

  • 1~31에 몰리기 쉬운 생일 번호 편향을 줄이기 위한 옵션입니다.
  • 한 게임에서 32~45 번호가 최소 3개 이상 포함되도록 유도합니다.

상세 설명

  • 예: 14-15 같은 인접 번호 쌍이 최소 1개 이상 들어가게 합니다.
  • 완전히 떨어진 숫자만 있는 조합을 피하고 패턴 다양성을 높입니다.

상세 설명

  • 현재 기준 인기 숫자로 분류한 7번을 후보군에서 제외합니다.
  • 자주 선택될 가능성이 높은 숫자를 피하고 싶을 때 사용합니다.

상세 설명

  • 홀수/짝수 비율이 2:4에서 4:2 범위 안에 들어오도록 제한합니다.
  • 한쪽(예: 홀수 6개)으로 과도하게 치우친 조합을 줄입니다.

상세 설명

  • 저번호(1~22)와 고번호(23~45)가 각각 2~4개가 되도록 맞춥니다.
  • 번호대가 한쪽으로 몰리는 조합을 줄이기 위한 옵션입니다.

상세 설명

  • 선택된 6개 번호 합계를 100~170 범위로 제한합니다.
  • 너무 낮거나 너무 높은 극단적 합계를 가진 조합을 줄입니다.

상세 설명

  • 같은 끝수(예: 3, 13, 23)가 3개 이상 몰리지 않도록 제한합니다.
  • 일의 자리 분포를 퍼뜨려 패턴 편중을 완화합니다.

상세 설명

  • 6개 전체가 일정 간격을 갖는 완전 등차수열 조합을 제외합니다.
  • 예: 3, 8, 13, 18, 23, 28 같은 직선형 패턴을 방지합니다.
